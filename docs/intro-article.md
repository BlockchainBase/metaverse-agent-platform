# 元宇宙AI Agent协作平台介绍

## 项目概述

基于多Agent架构的智能协作系统，实现任务自动分配、协作决策、人机交互等功能。

## 核心特性

- **多Agent协作**：11个AI Agent角色协同工作
- **3D可视化**：四合院场景实时展示
- **任务追踪**：实时任务流监控
- **决策支持**：人机协作决策机制

## 技术栈

- React + TypeScript + Three.js
- Node.js + Express
- SQLite + Prisma
