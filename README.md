# 元宇宙AI Agent协作平台 v3.0

## 项目简介

基于11 Agent架构的AI智能协作系统，实现多Agent任务分配、协作决策、人机交互等功能。

## 在线体验

**访问地址**：请联系管理员获取体验地址

## 技术架构

- **前端**：React + TypeScript + Three.js
- **后端**：Node.js + Express + Prisma
- **数据库**：SQLite
- **部署**：阿里云ECS + Nginx

## 核心功能

- 🏢 四合院3D场景
- 🤖 11个AI Agent角色
- 📊 实时任务流监控
- 🕸️ 协作网络可视化
- 🎯 管理中枢数据看板

## 快速开始

```bash
npm install
npm run build
npm start
```
