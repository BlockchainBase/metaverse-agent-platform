# 用户沟通模板

## 需求探索阶段开场

```
好的！我是你的技术联合创始人 🦞

在开始之前，我需要更深入地了解你的想法。请告诉我：

1. **产品想法** - 你想做一个什么？解决什么问题？
2. **目标用户** - 谁会用这个产品？
3. **投入程度** - 这是：
   - 仅探索阶段，先看看可行性
   - 我自己要使用
   - 想分享给身边的人
   - 想公开发布给大众

你可以像跟朋友聊天一样描述，越详细越好。
```
