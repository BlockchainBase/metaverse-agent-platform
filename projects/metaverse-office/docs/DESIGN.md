### 1.2 3D场景风格

**风格定位**: 科技感 + 简洁现代 + 轻微赛博朋克

**参考风格**:
- 《看门狗：军团》UI风格
- 《赛博朋克2077》科技感
- 苹果Vision Pro空间UI

**3D元素风格**:
- 建筑：低多边形(Low Poly) + 玻璃材质
- Agent：卡通渲染(Toon Shader) + 发光边缘
- 特效：粒子系统 + 霓虹光效

## 2. AI Agent 3D形象设计

### 2.1 AI教学秘书 🎓

**形象特征**:
- 人形机器人
- 头戴智能眼镜
- 手持发光书本/平板
- 配色：紫色系

**状态动画**:
```
Idle (空闲):
- 轻微呼吸起伏
- 书本悬浮旋转
- 紫色光环缓慢脉动

Working (工作中):
- 快速翻书动作
- 头顶出现数据流特效
- 紫色光环快速旋转

Busy (忙碌):
- 身体微前倾
- 多本书环绕
- 光环变红闪烁
```

**3D模型规格**:
- 高度: 1.7米 (人类比例)
- 面数: < 5000 (LOD0)
- 贴图: 1x 2K PBR

### 2.2 AI项目管家 📊

**形象特征**:
- 商务风格机器人
- 手持文件夹/图表
- 配色：青色系

**状态动画**:
```
Idle:
- 文件夹悬浮
- 图表缓慢旋转
- 青色光环

Working:
- 翻阅文件夹
- 数据图表弹出
- 进度环动画
```

### 2.3 AI设备助手 🔧

**形象特征**:
- 工程师风格
- 工具腰带
- 手持平板显示仪表盘
- 配色：橙色系

**状态动画**:
```
Idle:
- 工具悬浮
- 仪表盘缓慢扫描

Working:
- 维修动作
- 火花特效
- 警报灯闪烁(异常时)
```

### 2.4 AI企业顾问 💼

**形象特征**:
- 商务精英风格
- 公文包
- 数据环绕
- 配色：绿色系

### 3.2 场景元素设计

- **项目管道**: 3D透明管道，项目球体流动

- **会议区**: 项目讨论虚拟空间

- **设备地图**: 3D俯视图，设备实时状态
- **数据流**: 设备间数据传输粒子效果
- **告警区**: 异常设备红色闪烁

#### 5F - 产业孵化
- **企业工位**: 每个孵化企业独立空间
- **成长树**: 企业发展可视化（树状图）
- **投资墙**: 投融资数据展示

## 5. 动效设计

### 5.1 转场动画
- **楼层切换**: 电梯升降效果，3秒过渡
- **视角切换**: 平滑移动+旋转，2秒过渡
- **Agent聚焦**: 镜头推近+高亮边框

### 5.2 状态切换
- **状态变化**: 颜色渐变 + 粒子爆发
- **任务完成**: 粒子特效 + 成就弹出
- **异常告警**: 红色闪烁 + 脉冲波纹

### 5.3 环境动效
- **数据流**: 粒子沿管道流动
- **悬浮元素**: 正弦波上下浮动
- **光环**: 旋转+呼吸效果

## 6. 交互设计

### 6.1 鼠标交互
| 操作 | 效果 |
|------|------|
| 悬停Agent | 显示信息卡片 + 放大1.2x |
| 点击Agent | 聚焦视角 + 打开详情面板 |
| 拖拽场景 | 旋转视角 |
| 滚轮 | 缩放视角 |
| 双击空白处 | 重置视角 |

### 6.2 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| `1-5` | 快速切换楼层 |
| `Tab` | 切换聚焦的Agent |
| `Space` | 暂停/播放动画 |
| `F` | 全屏模式 |
| `ESC` | 退出全屏/关闭面板 |
