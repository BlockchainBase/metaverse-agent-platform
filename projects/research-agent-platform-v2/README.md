## 🚀 快速启动

### 1. 安装依赖

```bash
cd src/frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问: http://localhost:5173

### 3. 启动后端服务（在其他终端）

```bash
cd src/backend
npm install
npm run dev
```

## 🎮 功能特性

### 1. 元宇宙监控中心

- **8个AI Agent 3D形象** - 在四合院场景中走动
- **四阶段业务流可视化** - 市场→方案→研发→交付
- **实时协作展示** - Agent间消息流转动画
- **点击交互** - 查看Agent详情、任务状态

### 2. 多OpenClaw协作

- **消息路由** - OpenClaw ↔ OpenClaw 直接通信
- **任务协作** - 跨Agent任务分配与跟踪
- **状态同步** - 实时同步各Agent工作状态
- **飞书集成** - 移动端接收通知

### 3. Web管理后台

- **项目管理** - 四阶段流程跟踪
- **客户管理** - 线索与商机管理
- **财务管理** - 预算、成本、收款
- **任务看板** - 团队任务分配

## 🔧 配置说明

### OpenClaw接入

每个员工的OpenClaw需要配置:

```bash
# .env
AGENT_ID=market-agent
AGENT_NAME=AI市场专员
OWNER_NAME=市场经理
PLATFORM_API_URL=http://平台服务器IP:3001/api
```

### 飞书集成（可选）

```bash
FEISHU_APP_ID=cli_xxxxx
FEISHU_APP_SECRET=xxxxx
FEISHU_CHAT_ID=oc_xxxxx
```

## 🤝 协作模式示例

```
场景: 智慧校园项目推进

1. 刚哥（AI院长助理）:
   "推进智慧校园项目到下一阶段"

2. AI院长助理 → AI项目管家:
   "查询智慧校园项目当前状态"

3. AI项目管家回复:
   "阶段2已完成，可以进入阶段3"

4. AI院长助理 → AI方案架构师:
   "确认方案是否已评审通过"

5. AI方案架构师回复:
   "已通过，可以移交开发"

6. AI院长助理 → AI开发工程师:
   "智慧校园项目进入研发阶段，请准备开发环境"

7. 各Agent向各自真人汇报

8. 元宇宙空间展示:
   - 项目球从阶段2流向阶段3
   - 各Agent移动协作动画
   - 刚哥旁观整个流程
```
